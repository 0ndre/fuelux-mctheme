Usage

Via data-attributes

To enable the checkbox control without writing Javascript, add data-initialize="checkbox" to the parent of the checkbox input that you wish to initialize. Such elements that exist when $.ready() executes will be initialized. Any checkbox markup that is programatically created with data-initialize="checkbox" sometime after the initial page load will not immediately initialize. Rather, it will be initialized when the mouseover event is fired on it.

Via JavaScript

Call the checkbox control via JavaScript:

{% highlight js %}$('#myCheckbox').checkbox();{% endhighlight %}

Methods

$().checkbox();

Activates your content as a custom checkbox

.checkbox('check')

Checks the checkbox

{% highlight js %}$('#myCheckbox').checkbox('check');{% endhighlight %}

.checkbox('destroy')

Removes all functionality, jQuery data, and the markup from the DOM. Returns string of control markup.

{% highlight js %}$('#myCheckbox').checkbox('destroy');{% endhighlight %}

.checkbox('disable')

Disables the checkbox

{% highlight js %}$('#myCheckbox').checkbox('disable');{% endhighlight %}

.checkbox('enable')

Enables the checkbox

{% highlight js %}$('#myCheckbox').checkbox('enable');{% endhighlight %}

.checkbox('isChecked')

Returns true or false depending on whether the checkbox is checked

{% highlight js %}$('#myCheckbox').checkbox('isChecked');{% endhighlight %}

.checkbox('toggle')

Toggles the checkbox between checked/unchecked states

{% highlight js %}$('#myCheckbox').checkbox('toggle');{% endhighlight %}

.checkbox('uncheck')

Unchecks the checkbox

{% highlight js %}$('#myCheckbox').checkbox('uncheck');{% endhighlight %}
